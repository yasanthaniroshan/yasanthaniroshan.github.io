A real-time bus tracking system built with **Django**, pairing on-vehicle **ESP32 GPS units** with a web application that shows buses moving live on a Google Map.

## Key features

- **Live tracking** — GPS positions stream from the vehicles and update the map in real time
- **Schedule management** — routes and timetables administered through the dashboard
- **Rider-friendly dashboards** — passengers see where their bus actually is, not where it should be
